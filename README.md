# DunkenOrganizer
A GUI file organizer written in python.
It Automatically moves files in a selected folder into appropriate categories.
Such as:
1. Images
2. Archives
3. Documents
4. Others
# Usage on Linux and Mac:
```
python3 main.py
```

# Usage on Windows:
```
python main.py
```

Note:
If you want to organize other types files into different categories you can modify the categories.json file to include the category name and list of extensions.
# Example of adding more categories to categories.json:
```
{
    "Category Name 1": [".extension 1", ".extension 2", ".extension 3"],
    "Category Name 2": [".extension 1", ".extension 2", ".extension 3"]
}
```
